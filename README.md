## Setup
```shell
$ git clone https://github.com/jfarriagada/intro-redux.git
$ cd intro-redux
$ npm install
$ npm run build
```
En otra Terminal para correr desde un servidor o solo ejecutar index.html en el navegador
```shell
$ npm run watch
```
